Welcome to about TravelYatri

Our TravelYatri focuses on developing a system that automates the processes and activities of a travel and tourism agency. The purpose is to design a system using which one can perform all operations related to traveling and sight-seeing. TravelYatri is used to manage the details of Customer, Package Booking, Cancellation and Tourism places. It manages all the information about Users, Packages etc. The project is totally built at administrative end and thus only the admini is guaranteed the access to the backend database. The purpose of this project is to build an application program to reduce the manual work for managing Tourists, Booking, Places, etc.
